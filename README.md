# TaskFlow – React Todo Application

TaskFlow is a thoughtfully designed task management application built with React, focused on speed, usability, and clean user interactions.

Instead of being just another basic todo app, this project focuses on real-world task management workflows with instant updates, smooth interactions, and a responsive interface that works seamlessly across desktop, tablet, and mobile devices.

## Why I Built This

I wanted to build something simple on the surface but challenging from an engineering and UX perspective.

This project helped me explore:

- Component-driven architecture
- State management in React
- Controlled form handling
- Dynamic list rendering
- Responsive UI implementation
- Clean and maintainable code structure

## Features

✅ Create new tasks instantly  
✅ Mark tasks as completed  
✅ Edit existing tasks  
✅ Delete tasks  
✅ Real-time UI updates  
✅ Input validation  
✅ Keyboard support  
✅ Responsive design for all screen sizes  
✅ Clean and intuitive user experience
